# Taller de Github

## Descripción
El presente repositorio fue creado con el fin de comprender las herramientas proporcionadas por Visual Studio Code y Github, y el manejo de repositorios locales y remotos.

## Contenido
Algunos de los elementos utilizados fueron: 
- Commit
- Pull
- Push
- Edición de archivos desde Github y Visual Studio Code
- Creación de repositorios locales y remotos
